def authentication.views.GetDetails (   request)

Invoked when a GET request to "api/get-basic-details/" is made.

Returns a JSON response with keys 'success' and 'loggedin'. If user is logged in, then 'loggedin' is set to True and additional information of user namely 'username', 'email', 'first_name' and 'last_name' is also returned in the JSON response.

◆ LoginUser()

def authentication.views.LoginUser (   request)

Invoked when a POST request to "api/login/" is made.

Returns a JSON response with keys 'success' and 'message'. If user is successfully logged in, then 'success' is set to True and additional parameters namely 'token' and user details are returned. The token is used to identify the user in subsequent requests and must be present in the header as Authorization: "Token {TokenValue}"
